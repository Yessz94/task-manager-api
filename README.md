# task-manager-api
# Task Manager API - Python Flask

API REST desarrollada con Flask para gestionar tareas.

## Funciones

- Crear tareas
- Listar tareas
- Actualizar estado
- Eliminar tareas

## Tecnologías

- Python
- Flask
- JSON

## Ejecutar proyecto

Instalar dependencias:

pip install -r requirements.txt

Ejecutar servidor:

python app.py
